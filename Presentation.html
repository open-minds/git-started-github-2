Git WorkFlow

Git WorkFlow

Create a folder

$ mkdir myapp


Move to that Folder

$ cd myapp


Initialize git on that Folder

$ git init


Create a file

$ touch index.html

STATUS




git can tell you if your folder has some modified files (dirty)

$ git status

COMMIT


Snapshot of your work


HOW TO COMMIT ?


Create a folder

$ mkdir myapp


Add all the files in Staging area

$ git add --all


Take a snapshot (commit)

$ git commit -m "a message about this change"

DIFF

if git status tells you something changed, you can inspect exactly what changed :

$ git diff

LOG

Show commit history with

$ git log
